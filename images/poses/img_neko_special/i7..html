    New Rich Content Editor (RCE) 1/6

    The New Rich Content Editor (RCE) will be enabled across all courses on the University of Pennsylvania Canvas instance on January 6, 2021. The New RCE is an update to the existing text editor the provides an updated, condensed, and more intuitive interface. 

    For more information, visit the New Rich Content Editor guide. 

    Please direct any questions or concerns to canvas@pobox.upenn.edu
